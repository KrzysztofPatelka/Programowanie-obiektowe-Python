Opis programu "rankingtracks.py":
W programie wykorzystano zapis danych z plików „unique_tracks.txt” oraz „triplets_sample_20p.txt” do bazy danych Microsoft SQL Server 2017 Express Edition. Microsoft SQL Server 2017 został wybrany ze względu na to że był już zainstalowany na komputerze na którym testowany był program. Program realizuje zapis wskazanych wyżej plików do bazy danych oraz wyświetlanie informacje o artyście z największą liczbą odsłuchań oraz o 5 najpopularniejszych utworach.

Wyświetlenie informacji o artyście z największą ilością odsłuchań utworów. Działanie programu:
--MENU--
a - zapis pliku unique_tracks.txt do bazy danych
b - zapis pliku triplets_sample_20p.txt do bazy danych
c - artysta z największą liczbą odsłuchań
d - 5 najpopularniejszych utworów
x - wyjście z programu
Wprowadź (a/b/c/d/x): c
--
